How to use your store tools

Step-by-step walkthroughs for daily work: InvenSync, Oracle, Daily Report, POS Sold, Delivery, TransAct tools, and Store Data.

If you encounter missing data, upload issues, locked records, incorrect values, or any system problem, please create a ticket so the support team can review it properly. Include the store name, affected date, menu/tool, and a short description of the issue.

Walkthrough

InvenSync

  1. Select the correct inventory date at the top.
  2. Use Search or category buttons to find products faster.
  3. Follow each column from Beg through Delivery, Trans-In, BO, Adv, Trans-Out, Waste, CSI, Sold, and ending counts.
  4. See how Delivery, TransAct forms, Wastage, and POS Sold feed the inventory flow.
  5. Review calculated ending inventory, theoretical ending, quantity variance, and peso variance.
  6. Click End of Day when everything is checked and ready to finalize.
Start Walkthrough

Walkthrough

POS Sold

  1. Choose the report date before scanning or uploading.
  2. Select the POS Sold Excel file and click scan/upload.
  3. Review extracted products, quantities, sales, and discounts.
  4. If Additional Charge for Motif appears, complete the motif breakdown.
  5. Save or proceed so POS quantities reflect in InvenSync Sold.
Start Walkthrough

Walkthrough

Delivery

  1. Select the correct delivery date.
  2. Upload or scan the RSO file.
  3. Review product names and received quantities.
  4. Use manual add only for products not included in the RSO file.
  5. Click Save so delivery quantities reflect in InvenSync.
Start Walkthrough

Walkthrough

Daily Report

  1. Confirm the report date in the calendar.
  2. Enter POS sales, CI sales, TC, sales channels, spoilage, and discounts.
  3. Review the values before submitting.
  4. Click Submit Report.
  5. Confirm the send modal so the report is submitted to dashboards.
Start Walkthrough

Walkthrough

Oracle

  1. Open Oracle and check the selected ordering date.
  2. Review stock, sales average, and suggested order quantities.
  3. Use Daily Averages as a reference only; the cells are not editable.
  4. Review RSO / BO / POS tags when shown.
  5. Submit or export the data when needed.
Start Walkthrough

Walkthrough

TransAct Form

  1. Choose the transaction type and date.
  2. Check the generated control number.
  3. Select source/destination store and add products.
  4. Review product quantities and remarks.
  5. Save or submit the transaction so it can be tracked.
Start Walkthrough

Walkthrough

Trans-In

  1. Open Trans-In to view incoming transfers for your store.
  2. Select a pending transfer to review its product list.
  3. Check received quantities and short/over values.
  4. Use the action button to open the transfer details, review the items, and confirm or save the received quantities.
  5. Save the review so received stock can be reflected properly.
Start Walkthrough

Walkthrough

Trans-Out

  1. Open Trans-Out to view transfers sent by your store.
  2. Open a transfer record to review product and quantity details.
  3. Check status and remarks for tracking.
  4. Use the action button to view the full transfer details, including items, quantities, destination, status, and remarks.
  5. Use this page to verify outgoing stock movement.
Start Walkthrough

Walkthrough

Wastage

  1. Open Wastage to view wastage transfer records.
  2. Select a record to review product quantities and remarks.
  3. Check short/over details if applicable.
  4. Use the information to verify wastage reflected in reporting.
Start Walkthrough

Walkthrough

Store Data

  1. Select month and year to view historical store data.
  2. Review submitted Daily Report values and automated InvenSync values.
  3. Use the table to check sales, spoilage, discounts, and ending inventory.
  4. Export Excel when you need a file copy.
